# Lastream.Issues

This repository is for submitting issues and feature ideas for the Lastream app.

## About Lastream

Lastream is a powerful streaming application available on multiple platforms.

- **Website**: https://www.lastream.net/
- **iOS App**: https://apps.apple.com/us/app/lastream/id1483386771

## How to Use This Repository

If you've encountered a bug, have a feature request, or want to suggest improvements for Lastream, please create an issue in this repository. We appreciate your feedback and contributions to making Lastream better!
